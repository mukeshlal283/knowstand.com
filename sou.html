EASY WAY TO BUILT HEADER

Header is a main part of every and each website it is located at top of website. It basically contain a logo or name of website, navigation bars, search bar.

It is easy to make header through using html and css

HTML

<body> <header> <img src="foldername/imagename.extension"> <div class="navigation"> <ul> <li>HOME</li> <li>PAGE</li> <li>ABOUT</li> <li>CONTACT</li> </ul> </div> </header> </body>

CSS(style)

*{ padding: 0; margin: 0; } header{ width: 100%; height: 80px; background-color: grey; } header img{ float: left; height: 60px; padding: 10px 0; margin-left: 50px; } .navigation{ float: right; margin-right: 30px; } .navigation ul li{ display: inline-block; font-size: 20px; padding: 30px 20px; font-weight: bold; }

  • write html basic
  • write header tag
  • for display logo use <img> tag under image tag write folder name where image is kept and then write image name with extension (.jpg, .png)
  • for display navigation (ex-HOME, CONTACT, ABOUT, PAGES etc) take <navigation> tag
  • under navigation take <ul> tag
  • under li write home, contact, pages as you want to show
